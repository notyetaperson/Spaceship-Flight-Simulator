# 🌌 Spaceship Flight Simulator

A immersive 3D space flight simulator built with **Three.js**.

Explore vast procedurally generated environments, fly realistic spaceships, navigate around planets, asteroids, and celestial bodies in a beautiful open space experience.

[Play Online →](https://notyetaperson.github.io/Spaceship-Flight-Simulator/)

---

## ✨ Features

- Smooth spaceship flight mechanics
- Procedural planets and terrain
- Dynamic camera systems (cockpit + free cam)
- Asteroids, stars, and celestial objects
- Particle effects and engine trails
- Physics-based movement and collision
- Multiple camera views
- Sci-fi UI and controls

## 🎮 Controls

| Action              | Key / Input          |
|---------------------|----------------------|
| Thrust / Move       | W / S                |
| Strafe Left/Right   | A / D                |
| Roll                | Q / E                |
| Pitch & Yaw         | Mouse                |
| Boost               | Shift                |
| Brake               | Space                |
| Switch Camera       | C                    |
| Fire Weapons        | Left Click           |
| Pause / Menu        | Esc / P              |

## 🚀 How to Run Locally

### Windows (Easiest)
1. Download or clone the repository
2. Double-click `host.bat`
3. Open your browser and go to **http://localhost:8080**

### Using Python
```bash
